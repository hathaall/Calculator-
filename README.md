# Java Swing Calculator

A desktop calculator application featuring a clean, intuitive Graphical User Interface (GUI) built with Java's Swing library. This project provides standard calculator functionalities and is designed to be easily extensible for more advanced operations.

## Project Structure
This repository is configured as a NetBeans project and includes standard Apache Ant build files for easy compilation.

* `src/calculator/` - Contains the main Java source code and Swing UI components (JFrame, JPanel, etc.).
* `test/calculator/` - Contains unit tests for validating the underlying mathematical logic.
* `nbproject/` - NetBeans IDE project configuration files.
* `build.xml` - Apache Ant build script for compiling and packaging the application.

## Prerequisites
* **Java Development Kit (JDK)**: Version 8 or higher.
* **Apache Ant**: To compile and build the project via the terminal.
* **NetBeans IDE** (Optional): Highly recommended for editing the Swing GUI components via the Matisse UI builder.
